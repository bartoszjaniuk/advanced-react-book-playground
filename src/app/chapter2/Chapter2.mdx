# Key takeaways

- Komponent renderuje się, gdy jego obiekt elementu zmienia się, zgodnie z porównaniem Object.is przed i po re- renderowaniu
- Gdy elementy (komponenty) są przekazywane jako propsy do komponentu, a ten komponent uruchamia ponowne render poprzez aktualizację stanu, elementy, które są przekazywane jako propsy, nie będą ponownie renderować. (children as props)
